Welcome to the **Student Report Card System**!  
This is a simple Java console application designed to help teachers or schools generate student report cards, calculate totals, averages, and assign ranks based on performance.

---

## âœ¨ Features
- Input **school details** (school name, teacher name, grade, year).
- Enter **student records** including marks for:
  - English  
  - Math  
  - History  
  - Geography  
  - Science  
  - Programming
- Automatically calculates:
  - Total marks per student  
  - Rank (A, B, C, D, F) based on total score  
  - Subject totals and averages  
  - Count of students in each rank category
- Displays a **formatted report card table** with all details neatly aligned.

---

## ğŸ› ï¸ How It Works
1. The program prompts for **school details**.
2. You enter marks for each student (currently set to 2 students by default).
3. The system computes:
   - Individual totals  
   - Overall subject totals and averages  
   - Rank distribution
4. A **report card** is printed in a tabular format.
